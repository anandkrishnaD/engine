# 4-Stroke Engine Design

A mechanical CAD project focused on designing and validating a complete 4-stroke engine assembly, including motion analysis and interference checks.

---

## Overview
This project involves the design of a 4-stroke internal combustion engine using CAD tools. The goal was to model key engine components, assemble them into a functional system, and verify motion feasibility through simulation and clearance analysis.

---

## How it Works
The engine operates through the standard 4-stroke cycle: intake, compression, combustion (power), and exhaust. Each component—including the piston, crankshaft, connecting rod, and valve system—was modeled and assembled with proper constraints to replicate realistic mechanical motion. The system was tested through motion simulations to ensure smooth operation and correct timing.

---

## Components
- Piston  
- Crankshaft  
- Connecting Rod  
- Cylinder  
- Intake and Exhaust Valves  

---

## Tools Used
- Onshape (primary CAD modeling)  
- SolidWorks (design iterations and validation)  

---

## Key Features
- Complete 4-stroke engine assembly  
- Motion simulation of mechanical components  
- Interference and clearance validation  
- Iterative design improvements for alignment and performance  

---

## Technical Implementation
Each component was individually modeled and then assembled using CAD constraints to simulate real-world motion. Multiple design iterations were performed to improve alignment and eliminate interference. Motion analysis tools were used to validate the interaction between moving parts, achieving a fully functional system with 0% interference.

---

## Results
- Completed 4 design iterations  
- Achieved smooth motion across all components  
- Verified 0% interference between moving parts  
- Successfully demonstrated feasibility of the engine mechanism
